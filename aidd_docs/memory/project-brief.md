# Project Brief

What this project is, the problem it solves, and its domain language. The non-derivable "why", not the "how".

## What it is

- A personal portfolio website (Infuseting/Portfolio).

## Why it exists

- To showcase personal projects, skills, and professional experience in a visually appealing and performant web interface.

## Domain language

The terms a contributor must know to read the code.

| Term | Meaning |
| ---- | ------- |
| Astro | The web framework used to build fast, content-focused websites. |
| GSAP | GreenSock Animation Platform used for rich micro-animations and interactions. |

## Key features

- Static site generation for optimal performance.
- Rich animations using GSAP.
- Markdown/MDX content integration.
